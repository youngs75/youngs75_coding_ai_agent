## 중요 참고사항

### DashScope 국제판 API Key 구조
- 국제판(alibabacloud.com)은 모든 API Key가 **워크스페이스에 바인딩**
- generic URL(`dashscope-intl.aliyuncs.com`)은 워크스페이스 키와 호환되지 않음
- 반드시 워크스페이스 URL(`ws-<ID>.ap-southeast-1.maas.aliyuncs.com/compatible-mode/v1`)을 사용해야 함
- `core/model_tiers.py`에서 `api_base`를 명시적으로 전달하여 해결

### CLI 모드 MCP 동작
- Docker MCP(`localhost:3003`)의 mount root는 `/workspace` → 호스트 경로 set_workspace 불가
- CLI는 자동으로 로컬 MCP(`localhost:3013`)를 시작하여 호스트 workspace로 설정
- `CODE_TOOLS_MCP_URL` 환경변수를 런타임에 로컬 URL로 전환
- CLI 종료 시 로컬 MCP 프로세스 자동 terminate

### _execute_tool_safely 수정 영향
- MCP 도구 응답이 `[{'type': 'text', 'text': '...'}]` 리스트인 경우 텍스트 추출
- 문자열 응답은 기존과 동일하게 처리
- `written_files` 수집이 정상화되어 Phase별 파일 수 정확 표시
