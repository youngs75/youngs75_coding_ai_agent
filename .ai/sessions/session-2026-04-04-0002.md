## 중요 참고사항
- **패키지 구조**: import는 `youngs75_a2a`. pyproject.toml setuptools package-dir 매핑. 패키지 변경 후 `uv pip install -e .` 필수.
- **a2a 디렉토리**: `a2a_local/`로 리네이밍됨 (a2a-sdk 충돌 방지). import는 `youngs75_a2a.a2a`.
- **테스트 실행**: `source .venv/bin/activate && python -m pytest tests/ --ignore=tests/eval/ -v`
- **eval 테스트**: `python -m pytest tests/eval/ -v`
- **CLI 실행**: `youngs75-agent` 또는 `python -m youngs75_a2a.cli`
- **Docker CLI**: `cd docker && docker compose run --rm cli`
- **에이전트 팀**: tmux 세션 내에서 TeamCreate로 실행 (teammateMode: "tmux")
- **Langfuse 설정**: CLI_LANGFUSE_ENABLED=1 (기본 활성화), LANGFUSE_SECRET_KEY/PUBLIC_KEY 필요
- **.bashrc**: claude 명령어가 tmux 세션 자동 생성/연결하도록 alias 설정됨
